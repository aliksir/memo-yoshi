# MEMO-YOSHI 📝

**[English](README.md)**

> Claude Code への付箋。思いついたことを何でも書いておく場所。

## これは何？

Claude Code 用のメモツール — [猫軍団（neko-gundan）](https://github.com/aliksir/neko-gundan)マルチエージェントフレームワーク発。Markdownファイルに雑にメモを書いておくと、Claudeが読み込み・整理・アクション実行してくれる。

**困りごと:** Claudeが処理中に「あ、これ伝えたい」と思いつく。でも処理が終わる頃には忘れてる。URLをいっぱい読み込ませたいけど、1個ずつ開かせるためにとりあえずどこかにメモっておきたい。

**解決策:** いつでも開けるMarkdownファイル。URL、アイデア、TODO、愚痴、何でも書く。準備ができたらClaudeに読ませる。

## クイックスタート

```bash
git clone https://github.com/aliksir/memo-yoshi.git
bash memo-yoshi/install.sh
```

2ファイルがインストールされる:
- `~/.claude/MEMO-YOSHI.md` — メモファイル
- `~/.claude/commands/memo.md` — Claude Code スキル

## 使い方

| コマンド | 動作 |
|---------|------|
| `/memo` | MEMO-YOSHI.md をデフォルトエディタで開く |
| `/memo read` | メモを読み込んでセッションに取り込む |
| `/memo tidy` | 雑多なメモをカテゴリ分けして整理（URL、TODO、アイデア） |
| `/memo add Nodeバージョン確認` | タイムスタンプ付きでメモを追記 |
| `/memo act` | メモを解析してアクション実行（URL開く、確認実行など） |
| `/memo done` | 完了項目に `終わりヨシッ！` マーク（read/actでスキップされる） |

サブコマンド以外のテキストはそのままメモとして追記:

```
/memo APIドキュメント更新忘れずに
/memo https://気になる記事.com/あとで読む
/memo バグ: Safariでログインページ壊れる
```

## `/memo act` の動作

メモの中身を解析して、安全なアクションを自動実行する:

| メモの内容 | アクション |
|-----------|-----------|
| URL | ブラウザで開く |
| 「〜を確認して」「バージョン確認」 | 確認コマンド実行、結果をメモに書き戻す |
| 「〜を調べて」「リサーチ」 | Web検索して結果を書き戻す |
| 「〜を直して」「〜を実装して」 | **自動実行しない。** 確認を求める |
| ファイルパス | ファイルの存在確認、内容を要約 |

実行結果はタイムスタンプ付きで各メモの横に追記:

```markdown
- https://example.com → ✅ 開いた (2026-03-14 12:00)
- Nodeバージョン確認 → ✅ v22.5.0 (2026-03-14 12:01)
- ログインバグ修正して → ⏳ 依頼として認識。実行しますか？
```

## セッション開始時の自動読み込み（オプション）

毎セッション開始時にメモを自動で読み込みたい場合は、CLAUDE.md や `/takeover` ワークフローに追加:

```markdown
## セッション開始
- `~/.claude/MEMO-YOSHI.md` が存在すれば読み込む
```

## ルール

- **メモは自動削除されない。** `/memo tidy` は整理するだけ。削除はしない。
- **`/memo act` は破壊的操作をしない。** 実装・修正系の依頼は確認を求める。
- **`終わりヨシッ！` 付き項目はスキップされる。** `/memo read` と `/memo act` は完了済み項目を無視する。「全部読んで」と指定した場合のみ表示。
- 好きなテキストエディタで使える。専用アプリ不要。

## Claude Coworkでも使える

MEMO-YOSHIはClaude CodeとClaude Coworkの両方で動作する。Bashが使えない環境（Cowork等）では、ブラウザ起動やコマンド実行を手動案内に自動切替 — それ以外（read、tidy、add、done）はそのまま動く。

## 必要なもの

- Claude Code または Claude Cowork
- テキストエディタ（もう持ってるやつ）
